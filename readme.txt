How to use:
1. Download both the projects (auth0-m2m and auth0-spa)
2. Run 'npm install' under both the projects
3. Run 'npm start' under auth0-m2m
4. After auth0-m2m is started, run 'npm start' under 'auth0-spa'
5. Visit the URL: 'http:localhost:8080' in the browser
6. Click on 'Login' and follow the flow
7. The name and access-token of the logged in user should be displayed on the screen.
